What is git ?

  • Git is a version control system that has many uses

    • Some of git uses

      • keep track of what changes you made to your code
      • Sync. code between different people.
      • test changes in code without losing the original
      • revert back to old versions of code
      • can be used to push/pull code to/from online repositories like Github
    • But, What is Github ?
      • Github is a website that stores git repositories online, in order to facilitate collacoration of code.
        • Repository!!, repository is a sample place that keep track of code and all changes made to code.
  • Still confused between git and github?
    • git is the Service, github is the HUB. Think of git as the project WOKFLOW, and github is The Storage that holds the project files.

how to start with github and git ? " any commands starts with git is git Commands, others is Unix "

  1. Create a Github account, and create a new repository. GOOGLE IT, and copy the generated URL
    • GitHub Webpage
  2. Install git on your machine. GOOGLE it
  3. from your laptop use the terminal to navigate to the directory in which the repository will be downloaded from GitHub to the machine
    • cd </path/to/directory> => to moves between directories
    • git pull <url> => use the URL you copied from step 1. this command will clone "download" the remote repository to your local machine
  4. touch <filename.extension> => to create new file
  5. Open the file with your favourate texteditor, and write your code, Save IT
    • Example
      • touch index.html
  6. git add <file name> => add file(s) to staging area 'to be included in the next commit'
  7. git commit -m "message" => take a snapshot of the repository and save it with a message
  8. git push => will pushes the code to the remote repository
  9. if any updates to the code made from the Github repository, then we can use git pull to pull the changes locally. 'you must be inside project directory before pulling'
